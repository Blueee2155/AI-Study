Cyber Trace

Agent Execution Log

Terminal-style syntax-highlighted trace — zero CDN dependency.

agent-trace.log
agent run --model gpt-4 --verbose // Step 1: Planning → plan("Analyze user request") tokens: 1,204 | latency: 340ms // Step 2: Tool Call → tool.search("authentication middleware") ✓ Found 3 matches in src/middleware/ // Step 3: Code Generation → generate(context="auth.ts", mode="edit") tokens: 2,847 | latency: 1.2s ✓ Complete — 3 steps, 4,051 tokens, 2.1s total
Cyber-trace is for displaying CLI traces, agent logs, command execution records, and mixed-language pseudocode. Unlike code.html (which uses highlight.js CDN), syntax colors are applied via manual span classes (.kw, .fn, .str, .cmt, .var, .hl, .err). Unlike terminal.html (which shows simple command+output), cyber-trace shows structured, multi-line highlighted traces.
