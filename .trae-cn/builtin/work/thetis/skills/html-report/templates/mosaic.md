# Mosaic Template

## 1. Meta

- type: mosaic
- description: Card-based dashboard layout
- layout: single-column with multi-column card grids
- max-width: 1080px

***

## 2. Design Rules

### Allowed

- Use `--color-primary` (mapped to `--accent`) for accent: pills, numbered badges, strong text within stories, active indicators
- Use `--color-secondary` (mapped to `--accent2`) for soft backgrounds: pill backgrounds, quote panels, badge backgrounds, numbered circles
- Cards: `var(--bg2)` background + `1px solid var(--rule)` border + `border-radius: 10px`
- `border-radius: 10px` for cards and panels; `999px` for pills and circular badges only
- Hairline `1px solid var(--rule)` borders for cards and dividers
- Hierarchy through **font-size + font-weight + whitespace + grid layout**
- Line-height: titles 1.06–1.2, body text 1.5–1.6
- Monospace font (`var(--font-mono)`) for labels, dates, badges, crumbs — uppercase + letter-spacing for metadata
- Title and summary should follow the normal page content width;
- Grid layouts: prefer reusable auto-fit card grids; only force fixed column counts when the content semantics truly require it
- Semantic emphasis should also come from theme tokens; do not introduce extra hardcoded semantic colors

### Forbidden

- No `box-shadow` or `filter: blur()` — elevation is conveyed by border + background only
- No gradients except small avatar decorative gradients
- No hardcoded hex colors for theme-dependent elements — everything via `var(--*)`
- No decorative illustrations or hero images
- No sticky/fixed positioning

## 5. Components

### Meta Row (`.meta-row`)

```html
<!-- Flexible number of items — use as many or as few as needed -->
<div class="meta-row">
  <span><strong>{label}</strong>{value}</span>
  <!-- repeat as needed -->
</div>
```

Uses `grid` with `auto-fit` so items distribute evenly across rows.

### Hero (`.hero`)

```html
<section class="hero">
  <h1>{title}</h1>
  <p class="summary">{summary_text}</p>
</section>
```

Use the same content width as the page body. Do not render the title block as a narrow left column with a large empty area on the right.

### Auto-Fit Panel Grid (`.panel-grid`, `.problem`)

```html
<div class="panel-grid">
  <div class="panel">
    <p>{panel_text}</p>
  </div>
  <!-- optional: add more .panel blocks or pair with a .quote/.highlight callout -->
</div>
```


### Highlight Callout (`.quote`)

```html
<div class="quote">
  <div class="body">"{quote_text}"</div>
  <div class="author">— {author_name} · {author_title}</div>
</div>
```

### Compare Grid (`.compare-grid`)

```html
<div class="compare-grid">
  <div class="compare-card">
    <h3><span class="tick yes">✓</span>{left_list_title}</h3>
    <ul>
      <li>{left_item}</li>
    </ul>
  </div>
  <div class="compare-card">
    <h3><span class="tick no">×</span>{right_list_title}</h3>
    <ul>
      <li>{right_item}</li>
    </ul>
  </div>
</div>
```

### Data Table

```html
<div class="table-wrap">
  <table>
    <thead>
      <tr><th>{col_1}</th><th>{col_2}</th><th>{col_3}</th><th>{col_4}</th></tr>
    </thead>
    <tbody>
      <tr>
        <td>{val_1}</td>
        <td>{val_2}</td>
        <td class="target">{target_value}</td>
        <td>{val_4}</td>
      </tr>
    </tbody>
  </table>
</div>
```

### Numbered Item / Story (`.story`)

```html
<div class="stories">
  <div class="story">
    <div class="story-num">{number}</div>
    <div class="story-text"><strong>{item_title}</strong> {item_description}</div>
  </div>
</div>
```


### Auto-Fit Step Grid (`.timeline`)

```html
<div class="timeline">
  <div class="step">
    <span class="badge">{step_label}</span>
    <h4>{step_title}</h4>
    <div class="meta">{step_meta}</div>
    <ul>
      <li>{step_item}</li>
    </ul>
  </div>
</div>
```

### Open Item / Owner Row (`.question`)

```html
<div class="questions">
  <div class="question">
    <p>{entry_text}</p>
    <span class="assignee">
      <span class="avatar">{initials}</span>
      {owner_name} · {meta_text}
    </span>
  </div>
</div>
```

### Status Pill (`.pill`)

```html
<span class="pill draft"><span class="pill dot"></span>{status_text}</span>
```

### Chart Figure (`.chart-figure`)

Every ECharts chart MUST be wrapped in a `<figure>` with a visible `<figcaption>` title.

```html
<figure class="chart-figure">
  <figcaption>{chart_title}</figcaption>
  <div id="chart-{unique_name}" style="width:100%;min-height:400px"></div>
</figure>
```

### Diagram Figure (`.diagram`)

Every diagram (Mermaid, PlantUML, Graphviz, inline SVG) MUST be wrapped in a `<figure>` with a `<figcaption>` title.

```html
<figure class="diagram">
  <pre class="mermaid">
    flowchart LR
      A --> B --> C
  </pre>
  <figcaption>Figure X: {diagram_title}</figcaption>
</figure>
```
