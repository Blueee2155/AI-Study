# Forest Canopy

A natural and grounded theme inspired by lush woodland canopies and earthy tones.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#faf9f6` | Page canvas background (large area) |
| surface | `#f0efe8` | Secondary background (cards, callouts, code blocks) |
| text | `#2d4a2b` | Primary text color |
| text-muted | `#7d8471` | Secondary text (meta, captions, auxiliary info) |
| border | `#a4ac86` | Dividers, strokes |
| primary | `#3d6b39` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#e4e8d8` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | CrimsonPro | 400, 700 | CrimsonPro-Regular.ttf, CrimsonPro-Bold.ttf |
| mono | DMMono | 400 | DMMono-Regular.ttf |

## Best Used For

Sustainability reports, eco-friendly brands, wellness content, organic product showcases, nature-related presentations.
