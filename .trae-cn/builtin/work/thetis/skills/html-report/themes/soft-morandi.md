# Soft Morandi

A Japanese-inspired muted palette with sage green, warm cream, and dusty terracotta — soft, understated, and quietly elegant.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#FAF7F2` | Page canvas background (large area) |
| surface | `#ECE7E1` | Secondary background (cards, callouts, code blocks) |
| text | `#4B4B4B` | Primary text color |
| text-muted | `#8A817A` | Secondary text (meta, captions, auxiliary info) |
| border | `#D5CEC7` | Dividers, strokes |
| primary | `#7C9A92` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#F0E2D4` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | CrimsonPro | 400, 700 | CrimsonPro-Regular.ttf, CrimsonPro-Bold.ttf |
| mono | DMMono | 400 | DMMono-Regular.ttf |

## Best Used For

Lifestyle and wellness brands, interior design portfolios, cultural editorials, Japanese aesthetics, minimalist product showcases, calm and reflective long-form writing.
