# Modern Minimalist

A clean and contemporary theme built on refined neutrals for maximum clarity and elegance.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#fafafa` | Page canvas background (large area) |
| surface | `#f0f0f0` | Secondary background (cards, callouts, code blocks) |
| text | `#36454f` | Primary text color |
| text-muted | `#708090` | Secondary text (meta, captions, auxiliary info) |
| border | `#d3d3d3` | Dividers, strokes |
| primary | `#36454f` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#e8edf0` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | InstrumentSans | 400, 700 | InstrumentSans-Regular.ttf, InstrumentSans-Bold.ttf |
| mono | GeistMono | 400 | GeistMono-Regular.ttf |

## Best Used For

Architecture portfolios, design agencies, minimal branding, editorial layouts, high-end product showcases.
