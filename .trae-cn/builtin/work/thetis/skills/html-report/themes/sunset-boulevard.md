# Sunset Boulevard

A warm and vibrant theme inspired by golden-hour sunsets and Mediterranean coastlines.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#fdf8f3` | Page canvas background (large area) |
| surface | `#f9eedf` | Secondary background (cards, callouts, code blocks) |
| text | `#264653` | Primary text color |
| text-muted | `#5c7a84` | Secondary text (meta, captions, auxiliary info) |
| border | `#f4a261` | Dividers, strokes |
| primary | `#e76f51` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#fbe5d6` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | WorkSans | 400, 700 | WorkSans-Regular.ttf, WorkSans-Bold.ttf |
| mono | GeistMono | 400 | GeistMono-Regular.ttf |

## Best Used For

Creative agencies, travel presentations, lifestyle brands, warm storytelling decks, event pitches.
