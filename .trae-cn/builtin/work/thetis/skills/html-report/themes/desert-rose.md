# Desert Rose

A soft and sophisticated theme with dusty rose and warm clay tones evoking arid landscapes at dusk.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#fdf8f5` | Page canvas background (large area) |
| surface | `#f5ebe4` | Secondary background (cards, callouts, code blocks) |
| text | `#5d2e46` | Primary text color |
| text-muted | `#8c6b7a` | Secondary text (meta, captions, auxiliary info) |
| border | `#d4a5a5` | Dividers, strokes |
| primary | `#b87d6d` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#f2e0da` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | Lora | 400, 700 | Lora-Regular.ttf, Lora-Bold.ttf |
| mono | DMMono | 400 | DMMono-Regular.ttf |

## Best Used For

Beauty and fashion brands, wedding invitations, lifestyle blogs, interior design portfolios, feminine branding.
