# Midnight Galaxy

A dramatic and cosmic dark theme with deep purples and ethereal lavender highlights.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#2b1e3e` | Page canvas background (large area) |
| surface | `#382a50` | Secondary background (cards, callouts, code blocks) |
| text | `#e6e6fa` | Primary text color |
| text-muted | `#9a8fb5` | Secondary text (meta, captions, auxiliary info) |
| border | `#4a4e8f` | Dividers, strokes |
| primary | `#a490c2` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#3d3560` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | Outfit | 400, 700 | Outfit-Regular.ttf, Outfit-Bold.ttf |
| mono | GeistMono | 400 | GeistMono-Regular.ttf |

## Best Used For

Space and astronomy content, gaming presentations, creative agencies, entertainment pitches, futuristic branding, nightlife events.
