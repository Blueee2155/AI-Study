# Arctic Frost

A cool and crisp winter-inspired theme evoking clarity, precision, and icy freshness.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#fafcff` | Page canvas background (large area) |
| surface | `#edf3fa` | Secondary background (cards, callouts, code blocks) |
| text | `#1e3a5c` | Primary text color |
| text-muted | `#5e7d99` | Secondary text (meta, captions, auxiliary info) |
| border | `#c0cfe0` | Dividers, strokes |
| primary | `#4a6fa5` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#dce8f5` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | Jura | 300, 500 | Jura-Light.ttf, Jura-Medium.ttf |
| mono | IBMPlexMono | 400 | IBMPlexMono-Regular.ttf |

## Best Used For

Scientific presentations, data analytics dashboards, medical reports, precision engineering, winter-themed content.
