# Indigo Dusk

A refined light theme with deep indigo accents on cool gray, ideal for structured specification and planning documents.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#f5f7fa` | Page canvas background (large area) |
| surface | `#ffffff` | Secondary background (cards, callouts, code blocks) |
| text | `#0e1322` | Primary text color |
| text-muted | `#5a647a` | Secondary text (meta, captions, auxiliary info) |
| border | `#e2e6ee` | Dividers, strokes |
| primary | `#4a36e3` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#ece8ff` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | CrimsonPro | 400, 700 | CrimsonPro-Regular.ttf, CrimsonPro-Bold.ttf |
| mono | IBMPlexMono | 400 | IBMPlexMono-Regular.ttf |

## Best Used For

Product specs, design documents, technical proposals, project plans, roadmap decks, PRDs, and any structured planning deliverable.
