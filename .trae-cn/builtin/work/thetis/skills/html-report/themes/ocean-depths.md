# Ocean Depths

A professional and calming maritime theme that evokes the serenity of deep ocean waters.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#f1faee` | Page canvas background (large area) |
| surface | `#dff0ea` | Secondary background (cards, callouts, code blocks) |
| text | `#1a2332` | Primary text color |
| text-muted | `#4a6670` | Secondary text (meta, captions, auxiliary info) |
| border | `#a8dadc` | Dividers, strokes |
| primary | `#2d8b8b` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#d4eef0` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | IBMPlexSerif | 400, 700 | IBMPlexSerif-Regular.ttf, IBMPlexSerif-Bold.ttf |
| mono | IBMPlexMono | 400 | IBMPlexMono-Regular.ttf |

## Best Used For

Corporate presentations, financial reports, professional consulting decks, trust-building content.
