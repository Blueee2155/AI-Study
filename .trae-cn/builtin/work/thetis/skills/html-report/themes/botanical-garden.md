# Botanical Garden

A fresh and organic theme inspired by lush garden flora with warm accent blooms.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#f5f3ed` | Page canvas background (large area) |
| surface | `#ebe8df` | Secondary background (cards, callouts, code blocks) |
| text | `#2c3e2d` | Primary text color |
| text-muted | `#6b7c6b` | Secondary text (meta, captions, auxiliary info) |
| border | `#b8c9a9` | Dividers, strokes |
| primary | `#4a7c59` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#e2edd8` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | InstrumentSans | 400, 700 | InstrumentSans-Regular.ttf, InstrumentSans-Bold.ttf |
| mono | DMMono | 400 | DMMono-Regular.ttf |

## Best Used For

Botanical illustrations, garden planning, organic product branding, farm-to-table content, environmental reports, green initiatives.
