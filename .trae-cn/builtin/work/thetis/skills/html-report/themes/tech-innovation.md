# Tech Innovation

A bold and modern dark theme with high-contrast electric accents for cutting-edge technology presentations.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#1e1e1e` | Page canvas background (large area) |
| surface | `#2a2a2a` | Secondary background (cards, callouts, code blocks) |
| text | `#f0f0f0` | Primary text color |
| text-muted | `#a0a0a0` | Secondary text (meta, captions, auxiliary info) |
| border | `#3d3d3d` | Dividers, strokes |
| primary | `#0066ff` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#1a3a5c` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | BricolageGrotesque | 400, 700 | BricolageGrotesque-Regular.ttf, BricolageGrotesque-Bold.ttf |
| mono | JetBrainsMono | 400 | JetBrainsMono-Regular.ttf |

## Best Used For

Tech startups, software launches, innovation showcases, AI/ML presentations, digital transformation content, developer conferences.
