# Golden Hour

A rich and warm autumnal theme inspired by the golden light of late afternoon.

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| background | `#fdf9f3` | Page canvas background (large area) |
| surface | `#f5ebe0` | Secondary background (cards, callouts, code blocks) |
| text | `#4a403a` | Primary text color |
| text-muted | `#7a6e64` | Secondary text (meta, captions, auxiliary info) |
| border | `#d4b896` | Dividers, strokes |
| primary | `#d4910a` | Primary accent (links, heading decorations, quote lines) |
| secondary | `#faecd2` | Secondary accent (tag backgrounds, hover states, subtle highlights) |

## Typography

| Role | Font | Weights | Files |
|------|------|---------|-------|
| font | Lora | 400, 700 | Lora-Regular.ttf, Lora-Bold.ttf |
| mono | JetBrainsMono | 400 | JetBrainsMono-Regular.ttf |

## Best Used For

Literary content, warm editorial layouts, artisan brands, autumn campaigns, storytelling decks.
