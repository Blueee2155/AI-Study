# 模板角色归类表

请在「角色」列填入对应代号，多角色用逗号分隔。

## 角色代号

| 代号 | 角色                         | 说明           |
| -- | -------------------------- | ------------ |
| A  | 封面 (Cover)                 | Deck 第一页，定基调 |
| B  | 目录 (TOC)                   | 展示全局结构       |
| C  | 章节分隔页 (Section Divider)    | 章节过渡/呼吸页     |
| D  | 正文-文本 (Body: Text)         | 以文字为主        |
| E  | 正文-数据 (Body: Data)         | 数字/KPI 为核心   |
| F  | 正文-图表 (Body: Chart)        | 可视化图表        |
| G  | 正文-图解 (Body: Diagram)      | 流程/架构/脑图     |
| H  | 正文-时间线/计划 (Body: Timeline) | 进程/规划/步骤     |
| I  | 正文-对比 (Body: Comparison)   | 多选项/前后对比     |
| J  | 正文-图片 (Body: Visual)       | 以图片/视觉为主     |
| K  | 正文-代码 (Body: Code)         | 代码/终端展示      |
| L  | 正文-卡片网格 (Body: Cards)      | 多条目平铺        |
| M  | 号召行动 (CTA)                 | 激发行动/下一步     |
| N  | 结束页 (Closing)              | Deck 最后一页    |
